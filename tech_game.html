JavaScript Cheat Sheet

Variables:

var x = 10;
var name = "hello";
var arr = [1, 2, 3];

If / Else:

if (x > 5) {
    return true;
} else {
    return false;
}

For Loops:

for (var i = 0; i < arr.length; i++) {
    // do something with arr[i]
}

Common String Methods:

str.length
str.toUpperCase()
str.toLowerCase()
str.split("")
str.charAt(0)
str.includes("abc")
str.indexOf("abc")
str.slice(1)

Common Array Methods:

arr.push(item)
arr.pop()
arr.length
arr.indexOf(item)
arr.reverse()
arr.join("")
arr.sort()

Math:

Math.max(a, b)
Math.min(a, b)
Math.floor(x)
Math.ceil(x)
Math.abs(x)
x % 2 === 0  // even check

Operators:

+   // add
-   // subtract
*   // multiply
/   // divide
%   // remainder (modulo)
+=  // add and assign
*=  // multiply and assign
=== // strict equality
!== // strict inequality
