How to Play

  • Toggle Inputs: Click the input switches (A, B, C, D) to set them to 0 or 1.
  • Place Gates: Drag logic gates (AND, OR, XOR, NAND) from the toolbox onto the empty slots in the circuit.
  • Goal: Make the light bulb turn ON by choosing the correct gates and input combination.
  • Submit: Press Submit when you think your circuit is correct. Your score is based on accuracy and speed!

Logic Gate Cheat Sheet

Gate A=0, B=0 A=0, B=1 A=1, B=0 A=1, B=1
AND 0 0 0 1
OR 0 1 1 1
XOR 0 1 1 0
NAND 1 1 1 0

Quick Summary:

  • AND — Output is 1 only when both inputs are 1
  • OR — Output is 1 when at least one input is 1
  • XOR — Output is 1 when inputs are different
  • NAND — Opposite of AND (0 only when both are 1)
